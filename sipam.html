¿Por qué necesito hacer esto?

El SIPAM — Sistema Integral del Paciente Ambulatorio — fue diseñado originalmente para correr en terminales HP 700/92 de los años noventa. Nadie ha considerado valioso actualizarlo.

Tu Mac moderna no habla ese lenguaje por defecto. Esta guía instala las herramientas necesarias para que tu computadora nueva se comporte como aquella terminal antigua y puedas acceder al sistema.

Prerequisito
01 / 03

Descarga e instala iTerm2

iTerm2 es la aplicación de terminal que necesitamos. Es gratuita y de código abierto.

Descargar iTerm2

Una vez descargado: abre el .zip, arrastra iTerm a Aplicaciones y ábrelo. Si macOS pide confirmación, haz clic en Abrir.

  • Instalé iTerm2 y se abre correctamente
Confirma el paso anterior para continuar
Instalación automática
02 / 03

Instala todo con un clic

Este botón abrirá iTerm2 y ejecutará el script automáticamente. No necesitas escribir nada.

Ejecutar instalación
Confirmación de iTerm2: puede aparecer un diálogo antes de ejecutar. Haz clic en Run.
Contraseña: si el script la pide, escríbela y presiona Enter. No verás lo que escribes — eso es normal.

Verás mensajes con ✅ conforme avanza. Cuando termine aparecerá 🏁 Instalación completa.

Activación
03 / 03

Reinicia iTerm2 y conéctate

Cierra iTerm2 completamente con Cmd + Q y vuelve a abrirlo para que cargue el perfil SIPAM.

¡Listo! Verás el ícono de SIPAM en tu Dock — un pergamino como el que se muestra abajo. Haz clic en él para conectarte.
El ícono aparecerá en tu Dock automáticamente. Si no lo ves, búscalo en Aplicaciones y árralo al Dock manualmente.
  • Reinicié iTerm2 y hice clic en SIPAM desde el Dock
Confirma el paso anterior para continuar

C — Cómo usar las teclas de función

Para activar F7 (Continuar), presiona Fn + F7 simultáneamente en tu teclado de Mac.

En Macs con Touch Bar o teclados modernos, las teclas F1–F12 actúan como controles de brillo/volumen por defecto. Mantén Fn presionada para activar la función real.
